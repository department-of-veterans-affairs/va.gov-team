# Read My Profile

## Request

**Method**: GET

**Path**: /profile

**Parameters**:
None

**Headers**:
| Name | Type | Description |
|---|---|---|
|Authorization|JWT?|Token for access to the CRM API|
|secid|string|User security identifier|

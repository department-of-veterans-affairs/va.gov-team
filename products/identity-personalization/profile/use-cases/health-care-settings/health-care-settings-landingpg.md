# Overview 

**Health care settings** is a sub page in profile that encompasses health care related settings and preferences within profile. 

# Experience 

* The **Health care settings** page includes links to **Messages signature** and **Scheduling preferences** pages.  
  * **Health care contacts** is currently turned off due to technical issues.  
* Links appear in alphabetical order using the [Link \- Collection component](https://design.va.gov/components/link/collection).  
* There is a cross link to **My HealtheVet** based on user feedback.  
* The user must be LOA3 (identity verified) to see the page. If the user is not LOA3 verified, they will be directed to the **Sign-in information** page to verify their account.

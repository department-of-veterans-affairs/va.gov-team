## Requirements

1. 2-factor authentication status
   - If 2FA is not setup: prompt to add 2-factor
   - If 2FA is setup: hide component
2. Identity verification status
   - If account is LOA1: prompt to verify identity
   - If account is LOA3: language stating that identity is verified
3. Link to ID.me account management
4. Status of user's acceptance of MHV terms & conditions
   - If not accepted: link to [Terms and Conditions for Health Tools]()
   - If accepted: language stating health T&C have been accepted

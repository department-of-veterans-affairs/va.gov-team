# Initiative Brief: Expand Access to Scheduling

## Goals
The goal of this effort is to expand the options available for a Veteran to self-schedule a direct care appointment at facilities that use VistA. Increasing access to self-scheduling will enable Veterans to have better access to care.

## Github links
- [Epic](https://github.com/department-of-veterans-affairs/va.gov-team/issues/109449)

## Metrics
- Increase the number of appointments Veterans self-schedule
- Throughout this effort, we will be tracking the number of online appointments scheduled by Veterans using [VSSC reporting tools](https://app.powerbigov.us/groups/me/apps/057a6693-7190-4fd4-9491-f8a038913689/reports/efa6f7fb-5933-4009-aad9-a060d5da4be3/88fc5f3fdac93d3247f9?ctid=e95f1b23-abaf-45ee-821d-b7ab251ab3bf)

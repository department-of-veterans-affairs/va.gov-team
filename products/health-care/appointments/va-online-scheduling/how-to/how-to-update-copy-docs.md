# How to update copy docs

This is the step-by-step process. For in depth information, refer to GitHub's documentation: [Proposing changes to your work with pull requests](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests)

## Create a new pull request

1. After making changes, click `Commit changes...`

2. Select `Create a new branch for this change and start a pull request`

<img width="615" alt="Commit changes screen" src="https://github.com/department-of-veterans-affairs/va.gov-team/assets/2536801/eaa60493-efdb-4135-93f8-c6a4f0fd737f">

3. Name the pull request with the format `[username]-[ticketnum]-copy-update`

4. Click `Propose changes`

5. Review your changes in the following screen. Click `create pull request`.
  
## Edit or update a pull request
How to make changes to the code on the separate branch after you already created a pull request.

1. [Find your pull request](https://github.com/department-of-veterans-affairs/va.gov-team/pulls)

2. Click it to open it.

3. Click the `Commits` tab

<img width="817" alt="Commits tab selected" src="https://github.com/department-of-veterans-affairs/va.gov-team/assets/2536801/06993f5e-92ff-40ae-85ce-2699b3194466">

4. Click the commit to open it

5. Open the kebab menu to the right of the code changes view. Click 'edit file'

<img width="1027" alt="Screenshot 2023-07-03 at 8 48 16 AM" src="https://github.com/department-of-veterans-affairs/va.gov-team/assets/2536801/741b5916-a487-4bb9-8c12-c4950e5ec10e">

7. When you're done, commit the changes directly to the branch you've been working on

<img width="596" alt="Merge to current branch selected" src="https://github.com/department-of-veterans-affairs/va.gov-team/assets/2536801/3ea64ac9-f3f8-41ac-a9ed-f72d7fbe9982">

- **Step 1** - [Create a Research Plan](https://github.com/department-of-veterans-affairs/va.gov-team/blob/master/products/health-care/questionnaire/discovery/user-research/research-plan.md) including our hypothesis, methodology and request for participants(

- **Step 9** - Create [Personas](https://github.com/department-of-veterans-affairs/va.gov-team/blob/master/products/health-care/questionnaire/discovery/user-research/discovery-personas.md) that map our main users for the product
- **Step 10** - Create [Journey Map]( ) of the current and proposed workflow needs

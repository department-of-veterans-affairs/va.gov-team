# Onboarding Checklist
Nothing is required, but these steps should help you get started. You can start with reviewing the [accessibility documentation](https://github.com/department-of-veterans-affairs/va.gov-team/tree/master/products/disability/526ez/accessibility) for the 526ez

## Development & Tools

- [ ]  Complete the New VFS Team Member template tasks, including educational tasks and the tasks for Front & Backend engineers.
- [ ]  [Get your front and backend dev environment set up](https://depo-platform-documentation.scrollhelp.site/developer-docs/run-and-build-va-gov-locally)
- [ ]  Set up the [login workaround for frontend local dev](https://dsva.slack.com/archives/C053U7BUT27/p1725649845971869). It’s a timesaver for development.
- [ ]  Get access to a BrowserStack account for testing across multiple devices by requesting on #vfs-platform-support. There are limited seats.
- [ ]  Install the [Axe Devtools browser extension](https://www.deque.com/axe/browser-extensions/)
- [ ]  Install the [VA page checker Chrome extension](https://chromewebstore.google.com/detail/va-page-checker/bohcdnelkeimoooidokojkcjdaahjbkb?pli=1)
- [ ]  If you have a GFE Windows PC, you can use the VA’s license for JAWS & ZoomText Fusion. You have to submit a Service Now ticket. Go to the [508 Accessibility Catalog](https://gcc01.safelinks.protection.outlook.com/?url=https://yourit.va.gov/va?id%3Dsc_category_hcd%26sys_id%3Da7ff40ec1b352850a25d0d08ec4bcb6f%26catalog_id%3D1e105b9c1b062010a25d0d08ec4bcb03&data=04%7c01%7c%7c0257837f5ced4bc57d6b08d8d4cf5deb%7ce95f1b23abaf45ee821db7ab251ab3bf%7c0%7c0%7c637493334392909167%7cUnknown%7cTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7c1000&sdata=xhNSOizv1sEAXk9TdcrVukSIa7AnLtrhUJCk2RzO6PQ%3D&reserved=0). Select, “508 Accessibility Test Tools”, fill in your contact details, choose the tool name from the dropdown, describe your project and choose a date. For additional information contact [508toolssupportteam@va.gov](mailto:508toolssupportteam@va.gov)
- [ ]  Download or get access to AT & internet browsers. We test with the following combinations:
    - JAWS + Chrome
    - NVDA + Firefox
    - VoiceOver + Safari on macOS + iOS
    - Narrator + Edge
    - ZoomText/Fusion
    - TalkBack + Chrome on Android

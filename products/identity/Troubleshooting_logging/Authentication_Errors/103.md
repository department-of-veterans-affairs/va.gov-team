# ICN Mismatch

## Error code
`103`

## Title
ICN Mismatch

## Why it happens
This is due to an internal VA system reporting multiple identifiers for a user. This must be resolved manually and reported to the help desk.

## How to resolve the issue

1. Create a ticket with the help desk

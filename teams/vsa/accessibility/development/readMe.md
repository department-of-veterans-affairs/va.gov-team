# Accessibility Guidelines for Development

1. Establish e2e tests that include the axe checks.

1. Ensure engineering knows how to customize tests to ensure hidden and dynamic content is also evaluated.

1. Define workflow processes to evaluate 508 compliance and accessibility during development.
